## Description

[GDTF](https://gdtf-share.com) (General Device Type Format) is a standardized format for a digital representation of any lighting fixture.

From the [website](https://gdtf-share.com/landing/pages/about.php):
> The file format was developed to be an open standards technology to facilitate interoperability and data exchange between manufacturers and their designer base. Manufacturers involved in the entertainment design, production, and performance industry are welcomed to support GDTF & MVR and our goal to unify the industry.
